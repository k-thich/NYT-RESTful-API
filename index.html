NY Times RESTful API

Get articles

Returns a list of all articles sorted in alphabetical order

HTPP Request: http://127.0.0.1:8080/articles

Get authors

Returns a list of all known authors sorted in alphabetical order

HTPP Request: http://127.0.0.1:8080/authors

Get short urls

Returns a list of all short urls sorted and grouped by publish date

HTPP Request: http://127.0.0.1:8080/short_urls

Get hyperlinked article images

Returns a list of all articles as hyperlinked thumbnail images

HTPP Request: http://127.0.0.1:8080/articles_images

Get des_facet tags

Returns a list of des_facet tags displayed as tag cloud

HTPP Request: http://127.0.0.1:8080/articles?field=des_facet

Get article details

Returns the details of the article index given

If no index is provided, the index is defaulted to 0

If the given index is invalid, nothing is returned

Example HTPP Request: http://127.0.0.1:8080/articles/0
